# 🏠 House Price Prediction App

## 📌 Project Overview
This project is a Machine Learning application that predicts house prices based on input features such as area.

## ⚙️ Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- Streamlit

## 🧠 Model Used
Linear Regression model is used to predict house prices based on the given dataset.

## 📊 Features
- User-friendly interface using Streamlit
- Real-time prediction
- Simple and fast model

## ▶️ How to Run the Project

1. Install required libraries:
   pip install -r requirements.txt

2. Run the Streamlit app:
   streamlit run app.py
